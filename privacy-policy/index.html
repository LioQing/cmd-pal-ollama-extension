Privacy Policy

For LLM Extension for Command Palette

What User Information Does The Developer Collect?

LLM Extension for Command Palette (hereinafter referred to as the app) does not collect and send any user information to the developer of the app.

Local Configurations

The app stores user configurations locally on the device. These include settings, preferences, and API keys used to communicate with external service providers as specified by the user on the app. This information is stored locally and is not shared with the developer's servers.

External Service Providers

The app communicates with external service providers that are specified by the user on the app, which may collect user information. If the user wish to know what user information are collected by these external service providers, please visit the privacy policy of the corresponding services as selected by the user.

Contact

If you have any questions or concerns, you may contact the developer via email at lio@lioqing.com.
